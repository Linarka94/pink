### Installation

Install [Node.js](https://nodejs.org/) to run project and open folder with project in terminal. Install the dependencies and devDependencies, then build the project :rocket:
```sh
$ cd pink
$ npm install
$ gulp build
```

#### Commands
* Use `gulp` to run server in dev mode.
